## 🚀 Premiers Pas

### Installation
Markify fonctionne directement dans votre navigateur. Aucune installation n'est requise :

1. Téléchargez ou clonez le projet
2. Ouvrez le fichier `index.html` dans votre navigateur
3. Commencez à éditer vos messages Discord !

### Configuration Initiale
- **Langue** : Cliquez sur "FR" pour basculer entre Français et Anglais
- **Thème** : Cliquez sur 🌙/☀️ pour basculer entre thème sombre et clair
- L'application mémorise vos préférences automatiquement

## 🎨 Interface Utilisateur

Markify utilise une interface en 3 colonnes optimisée pour l'édition de messages Discord :

### Colonne Gauche : Sélecteur d'Émojis
- **Recherche** : Champ de recherche en haut
- **Catégories** : Onglets pour organiser les émojis
- **Grille d'émojis** : Cliquez pour insérer
- **Astuce drapeaux** : Info sur la combinaison d'émojis régionaux

### Colonne Centrale : Éditeur de Messages
- **Zone de texte** : Éditeur principal avec coloration syntaxique
- **Statistiques** : Compteurs de caractères (Standard/Nitro)
- **Boutons d'action** : Copier et Réinitialiser
- **Aperçu Discord** : Prévisualisation en temps réel

### Colonne Droite : Styles et Polices
- **Styles Markdown** : Boutons pour appliquer les styles Discord
- **Polices Unicode** : Collection de polices spéciales
- **Avertissement** : Rappel sur la compatibilité Unicode/Markdown

## ✏️ Édition de Messages

### Saisie du Texte
1. Cliquez dans la **zone de texte centrale**
2. Tapez votre message normalement
3. Les **codes Discord** sont automatiquement détectés et protégés :
   - Canaux : `#general` → `<#123456789>`
   - Utilisateurs : `@user` → `<@123456789>`
   - Rôles : `@role` → `<@&123456789>`

### Application des Styles
1. **Sélectionnez le texte** à styliser
2. **Cliquez sur un bouton de style** dans la colonne droite
3. Le style s'applique automatiquement avec les bonnes syntaxes Discord

### Aperçu en Temps Réel
- La section **"Aperçu Discord"** montre exactement ce que verra un utilisateur Discord
- Mise à jour automatique à chaque modification
- Styles visuels identiques à Discord

## 😊 Utilisation des Émojis

### Recherche d'Émojis
- Tapez dans le **champ de recherche** en haut de la colonne gauche
- Recherche par **nom anglais** ou **mot-clé français**
- Exemples : "smile", "sourire", "heart", "coeur"

### Catégories d'Émojis
- **Smileys** : Expressions faciales
- **People** : Gestes et personnes
- **Animals** : Animaux
- **Food** : Nourriture et boissons
- **Activities** : Activités et sports
- **Travel** : Voyages et lieux
- **Objects** : Objets
- **Symbols** : Symboles
- **Flags** : Drapeaux

### Insertion d'Émojis
- **Clic simple** : Insère l'émoji à la position du curseur
- **Double-clic** : Insère et copie dans le presse-papiers
- Les émojis s'intègrent parfaitement avec le texte et les styles

### Astuce Drapeaux
Vous pouvez créer des drapeaux de pays en combinant deux émojis de lettres régionales :
- 🇫 + 🇷 = 🇫🇷 (France)
- 🇺 + 🇸 = 🇺🇸 (États-Unis)
- 🇯 + 🇵 = 🇯🇵 (Japon)

## 🎯 Styles Markdown

### Styles de Base
- **Gras** : `**texte**` → **texte**
- **Italique** : `*texte*` → *texte*
- **Souligné** : `__texte__` → __texte__
- **Barré** : `~~texte~~` → ~~texte~~
- **Spoiler** : `||texte||` → ||texte||
- **Code inline** : `` `code` `` → `code`

### Titres
- **Titre 1** : `# texte` → # texte
- **Titre 2** : `## texte` → ## texte
- **Titre 3** : `### texte` → ### texte

### Blocs Spéciaux
- **Citation** : `> texte` → > texte
- **Citation multi-ligne** : `>>> texte` → >>> texte
- **Bloc de code** : ```code``` → ```
code
```
- **Liste** : `- item` → - item

## 🔤 Polices Unicode

### Polices Disponibles
1. **Bold** - Texte en gras Unicode
2. **Italic** - Texte en italique Unicode
3. **Bold Italic** - Gras et italique
4. **Sans Serif** - Police sans-serif
5. **Sans Bold** - Sans-serif gras
6. **Monospace** - Police à chasse fixe
7. **Double** - Caractères doublés
8. **Script** - Police scriptée
9. **Fraktur** - Police gothique
10. **Fullwidth** - Caractères pleine largeur
11. **Small Caps** - Petites capitales
12. **Upside Down** - Texte à l'envers
13. **Wavy** - Texte ondulé
14. **Zalgo** - Effet glitch
15. **Aesthetic** - Style esthétique
16. **Strikethrough** - Barré Unicode

### Utilisation des Polices
1. **Sélectionnez le texte**
2. **Cliquez sur une police Unicode**
3. ⚠️ **Important** : Les polices Unicode ne se combinent pas avec le Markdown Discord

### Conseils pour les Polices
- Utilisez **Monospace** pour un look technique
- **Script** et **Fraktur** pour un style élégant
- **Zalgo** pour des effets spéciaux (attention à la lisibilité)
- Testez toujours l'aperçu avant d'envoyer

## ⚡ Fonctionnalités Avancées

### Comptage de Caractères
- **Standard** : 2000 caractères maximum
- **Nitro** : 4000 caractères maximum
- **Lignes** : Nombre de lignes dans votre message
- Indicateurs visuels quand vous approchez des limites

### Gestion du Presse-Papiers
- **Copie automatique** : Bouton "Copier le message"
- **Fallback** : Méthode alternative si l'API clipboard échoue
- **Notifications** : Confirmation de copie réussie

### Thèmes et Accessibilité
- **Thème sombre** : Par défaut, facile sur les yeux
- **Thème clair** : Alternative pour environnements lumineux
- **Contraste élevé** : Bonne lisibilité dans tous les modes
- **Responsive** : Fonctionne sur desktop et mobile

### Internationalisation
- **Français** : Interface complète en français
- **Anglais** : Interface complète en anglais
- **Basculement** : Bouton en haut à droite
- **Persistance** : Mémorisation de la langue choisie

## 💡 Astuces et Conseils

### Optimisation des Messages
- **Combinez les styles** : `**__texte__**` pour gras + souligné
- **Émojis stratégiques** : Utilisez-les pour attirer l'attention
- **Citations pour réponses** : Idéal pour les discussions
- **Blocs de code** : Parfaits pour partager du code ou des commandes

### Productivité
- **Raccourcis clavier** : Ctrl+A pour tout sélectionner
- **Copie rapide** : Double-clic sur émoji pour copier
- **Réinitialisation** : Bouton reset pour repartir de zéro
- **Recherche émoji** : Mots-clés français supportés

### Bonnes Pratiques Discord
- **Limites de caractères** : Respectez les 2000/4000 caractères
- **Lisibilité** : Évitez trop d'émojis ou de styles
- **Accessibilité** : Pensez aux utilisateurs avec daltonisme
- **Modération** : Utilisez les spoilers pour le contenu sensible

### Combinaisons Puissantes
```
**__Titre Important__**
> Citation élégante
`code inline` et ```blocs de code```

```

## 🔧 Dépannage

### Problèmes Courants

#### La copie ne fonctionne pas
- **Cause** : API clipboard bloquée par le navigateur
- **Solution** : Autorisez l'accès au presse-papiers ou utilisez le fallback automatique

#### Les styles ne s'appliquent pas
- **Cause** : Aucun texte sélectionné
- **Solution** : Sélectionnez d'abord le texte à styliser

#### L'aperçu ne se met pas à jour
- **Cause** : Problème de synchronisation
- **Solution** : Actualisez la page ou réinitialisez l'éditeur

#### Émojis qui ne s'affichent pas
- **Cause** : Police système incomplète
- **Solution** : Utilisez un navigateur moderne avec support Unicode complet

### Support Technique
Si vous rencontrez un problème :
1. Vérifiez que vous utilisez un navigateur moderne (Chrome, Firefox, Edge)
2. Actualisez la page
3. Essayez en mode incognito
4. Contactez le support sur Discord : [REDIOUS Server](https://discord.gg/6b4NYRNsRC)

### Performances
- **Chargement initial** : Quelques secondes pour charger tous les émojis
- **Utilisation** : Fluide même avec de longs messages
- **Mémoire** : Optimisé pour ne pas surcharger le navigateur
