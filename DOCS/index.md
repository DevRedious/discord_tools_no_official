## 📖 Aperçu du Projet

Markify est un outil web gratuit pour créer des messages Discord stylisés avec :

- **Éditeur Markdown** complet (gras, italique, titres, blocs de code, etc.)
- **Sélecteur d'émojis** avec recherche dans 9 catégories
- **Polices Unicode** spéciales (15 polices disponibles)
- **Aperçu Discord** en temps réel
- **Protection automatique** des mentions et canaux
- **Interface bilingue** (Français/Anglais)
- **Thèmes sombre/clair** adaptatifs

## 🛠️ Technologies

- **Frontend** : HTML5, CSS3, JavaScript ES6+
- **Architecture** : Modules ES6, séparation des responsabilités
- **UI/UX** : Design responsive, accessibilité WCAG 2.1
- **Performance** : Optimisé pour le web moderne
